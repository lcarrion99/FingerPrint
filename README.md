# Java Fingerprint File Processor

## Description
Java program that reads fingerprint data from a file, stores it in a 2D array,
and compares fingerprint images pixel by pixel.

## Concepts Used
- File Input (Scanner + File)
- Object-Oriented Programming
- 2D Arrays
- Constructors
- Method Overloading
- Data Comparison Logic

## What I Learned
How to process structured data files and compare complex data sets.
